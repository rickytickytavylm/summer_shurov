# Трезвое лето с Шуровым — лендинг

Статичный лендинг челленджа. Без сборки: HTML + CSS + JS. Быстрый, mobile-first, в стиле iOS.

## Запуск

Никакого сервера не нужно — просто открой `index.html` двойным кликом в браузере.

## Структура

- `index.html` — разметка всех секций из ТЗ.
- `styles.css` — стили (iOS, летняя палитра, адаптив, мобильный tab bar).
- `script.js` — навигация, UTM, аналитика, рендер видео и соцсетей.
- `videos.js` — **данные видео участников** (добавление без правки кода).
- `favicon.svg` — иконка.
- `assets/` — картинки и видео (см. ниже, сейчас плейсхолдеры).

## Как добавить пост участника (лента Instagram)

1. Положи превью в `assets/posts/` — например `01.jpg`, `02.jpg` …
2. Открой `videos.js` и добавь объект в `items`:

```js
{
  author: "Имя Фамилия",
  handle: "@nickname",
  network: "Instagram",
  videoUrl: "https://...ссылка на пост/ролик...",
  authorUrl: "https://...профиль...",
  caption: "Короткая подпись",
  date: "2026-07-10",
  status: "опубликовано",
  pinned: false,
  thumb: "assets/posts/07.jpg"
}
```

Если файла превью нет — покажется зелёный плейсхолдер с подсказкой.

## Аналитика (перед публикацией)

1. **Яндекс.Метрика**: вставить счётчик в `<head>` `index.html` и задать `window.YM_COUNTER_ID = XXXXXX;` до подключения `script.js`. Цели настраиваются как `reachGoal` по именам событий (см. `data-analytics` ниже).
2. **Google Analytics (GA4)**: вставить gtag-сниппет в `<head>`. События уйдут через `gtag('event', ...)`.
3. **UTM**: проставляются автоматически на ссылки с атрибутом `data-utm`. Значения — в `script.js`, объект `UTM`.

### Имена событий (цели аналитики из ТЗ)
